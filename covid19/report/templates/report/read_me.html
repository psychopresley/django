Hi, there! This page is supposed to be just a cool way to visualize some aspects of the coronavirus outbreak, so you won't find here any deep analysis, modelling or advanced machine learning estimation of the near future. I will acknowledge here most of the tools that made possible to build this page and also provide some links if you're interested in a more deep analysis of the pandemics.

All graphics on this page were build with Flourish Studio. It's a very interesting web based visualization app with the option of creating storytellers, which is a cool way to present your data. As an example you can check my own storytelling for this project here. So, if you are interested in dive deep in the numbers here it is a few pages I recommend you to visit:

  • Original data source: All rights reserved to Johns Hopkins University
  • JHU visual dashboard. That's the most confident page to get real time data: JHU CSSE
  • If you're interested in checking my work just visit my project repository at github: project repository
  • Also, my own visual dashboard, build with Tableau public. There you will find a deeper analysis of the pandemics with much more visualization options. It's worth to take a look: The Covid-19 Dashboard
  • I also made available a dataset with the compiled numbers. It has some saved queries that can be run and provide you with cool information about the pandemics. Feel free to explore it! Tracking the covid-19


This page is powered by Django 3.5 and Python 3.9. I use Booststrap 5 for styling. The available data is collected from the original github repository and cleaned to build the dataset that provides all the information presented here. The dataset is updated daily.
